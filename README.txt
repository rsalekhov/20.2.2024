Документация по Телеграм-боту для изучения английских слов
Описание программы
Этот телеграм-бот предоставляет возможность изучения английских слов с использованием карточек. Пользователи могут добавлять новые слова, удалять их и проверять свои знания.

Требования
- Python 3.11 (или выше)
- Библиотека Telebot
- База данных PostgreSQL

20.02.2024.py - основной файл проекта
config.py - файл с конфигурациями (включая токены и адреса таблиц)

/start или /cards: Запускает процесс обучения английскому языку.
/add_word: Добавляет новое слово для обучения.
/delete_word: Удаляет слово из обучения.

Структура базы данных
user_info: Таблица, идентификации пользователя по ID
words_table: Таблица с основными словами для обучения.
other_words: Таблица с неправильными вариантами ответов
user_words*IDпользователя*: Таблица, созданная на базе words_table для каждого уникального пользователя, с которой он может работать.
